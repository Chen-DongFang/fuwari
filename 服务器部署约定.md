## 服务管理命令

### tute-order（进程名仍为tute-finance）
```bash
# 通过Supervisor管理（推荐）
supervisorctl status              # 查看状态
supervisorctl restart tute-finance  # 重启
supervisorctl stop tute-finance     # 停止
supervisorctl start tute-finance    # 启动
supervisorctl reread && supervisorctl update  # 重载配置

# 查看日志
tail -f /opt/apps/tute-finance/app.log
tail -f /opt/apps/tute-finance/error.log

# Supervisor配置文件
cat /etc/supervisor/conf.d/tute-finance.conf
```

# 启动

## Nginx配置
```nginx
# /etc/nginx/sites-available/yunxing.fun
server {
    listen 80 default_server;
    server_name _;

    location / {
        proxy_pass http://127.0.0.1:8080;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
    }

    location ~* \.(js|css|png|jpg|jpeg|gif|ico|svg|woff|woff2|ttf|eot)$ {
        proxy_pass http://127.0.0.1:8080;
        proxy_set_header Host $host;
        expires 30d;
        add_header Cache-Control "public, immutable";
    }
}
```

## 数据清除脚本
需要清除业务数据时执行（如客户试用前重置环境）：
```bash
# 停止服务
supervisorctl stop tute-finance

# 执行清理（Python脚本）
cd /opt/apps/tute-finance
source venv/bin/activate
python -c "
import sqlite3
conn = sqlite3.connect('instance/tute_order.db')
c = conn.cursor()
for t in ['audit_logs','reminders','attachments','payment_items','payments','order_items','order_staff','orders','customers']:
    c.execute('DELETE FROM ' + t)
conn.commit()
conn.close()
print('业务数据已清除')
"

# 清除附件
rm -f /opt/apps/tute-finance/uploads/*.jpg /opt/apps/tute-finance/uploads/*.jpeg /opt/apps/tute-finance/uploads/*.png /opt/apps/tute-finance/uploads/*.xlsx

# 启动服务
supervisorctl start tute-finance
```
